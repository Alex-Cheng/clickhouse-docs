# Integrations

ClickHouse integrations have these functional categories:
- Language Client
- Data Ingestion
- Data Visualization
- SQL Client

and, there are different support categories:
- Core
- Partner
- Community

The tables below link to details about the integrations, and are broken down by the support categories.
